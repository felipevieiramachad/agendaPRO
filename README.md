# AgendaPRO

Aplicação simples de gerenciamento de tarefas (To-Do List) feita com **HTML, CSS e JavaScript puro**. Permite adicionar, editar, concluir e excluir tarefas de forma prática e intuitiva.

## Funcionalidades

* ✅ Adicionar tarefas com título e descrição
* ✏️ Editar tarefas existentes
* ✔️ Marcar tarefa como concluída
* 🔄 Marcar tarefa como pendente
* ❌ Excluir tarefas

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

📁 projeto

## ▶️ Como executar

1. Baixe ou clone este repositório
2. Abra o arquivo `index.html` no navegador

Não é necessário instalar nenhuma dependência.

## Como usar

1. Digite um título (obrigatório)
2. (Opcional) Adicione uma descrição
3. Clique em **Adicionar**
4. Use os botões para gerenciar a tarefa:

   * **Concluir** → marca como finalizada
   * **Pendente** → volta para não concluída
   * **Editar** → altera título/descrição
   * **Excluir** → remove a tarefa

## ⚠️ Regras do sistema

* Não é permitido criar tarefas com o mesmo título
* O título não pode estar vazio

## Estados visuais

* 🟢 **Concluída** → tarefa finalizada (com risco no texto)
* 🟡 **Pendente** → tarefa ainda em aberto
